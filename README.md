# Jazhem Hamid Portfolio

A modern dark portfolio website for Jazhem Hamid, built to support applications for software developer, AI automation, and agentic development roles on OnlineJobs.ph and similar platforms.

The site positions Jazhem as a Software Developer focused on AI Automation, Agentic Workflows, RAG Systems, and Practical SaaS Tools.

## Sections

- Hero section with clear positioning
- About section
- Featured projects
- Skills
- Process / how I build
- Contact section

## Featured Projects

- ClientOps AI Dashboard
- KnowledgeBase RAG Assistant
- AutoFlow Agent
- Skin Disease Cam

Project buttons currently use placeholder links (`#`) for View Project and GitHub.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- CSS animations and responsive layout

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

Create a production build:

```bash
npm run build
```

Run the production server after building:

```bash
npm run start
```
